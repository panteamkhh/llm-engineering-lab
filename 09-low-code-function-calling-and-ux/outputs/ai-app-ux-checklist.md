# AI Application UX Checklist

- [ ] AI involvement clearly disclosed to the user
- [ ] Limitations and uncertainty communicated (e.g., "may be inaccurate")
- [ ] Sources/citations shown for grounded (RAG) answers
- [ ] User can edit, regenerate, or undo an AI-driven action
- [ ] Escalation path to a human exists
- [ ] Suggested prompts/examples reduce blank-page anxiety
- [ ] Feedback mechanism (thumbs up/down or correction) present
- [ ] Loading/streaming states match actual latency (function calls, image gen)

Reusable pattern: code/function_calling_dispatcher_from_scratch.py (function registry)
